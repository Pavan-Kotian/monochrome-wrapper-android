I built a simple Android app for Monochrome 🎵

Hey everyone,

I’ve been using https://monochrome.tf/ for a while and really liked the clean UI and music experience, so I decided to turn it into a basic Android app using WebView.

It’s lightweight, fullscreen, and feels more like a native app compared to using it in a browser. I also added small improvements like smoother navigation and better playback handling.

⚠️ Note: The download feature is currently not working inside the app. I’m actively working on fixing it.

🙏 Full credit goes to the Monochrome Music team for building this amazing FLAC music streaming site. I only created a wrapper app around their website.

This is mainly for personal use and learning purposes, but I thought some of you might find it useful too.

Would love your feedback

Thanks!
